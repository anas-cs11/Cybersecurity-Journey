# Cybersecurity Tools & Programming Languages
## Role of Tools in Cybersecurity
Cybersecurity tools help professionals monitor, detect, and respond to threats efficiently.They reduce manual effort and improve the ability to handle large-scale attacks in real time.
## Categories of Security Tools
Security tools are used for different purposes such as monitoring, analysis, testing, and defense.Common categories include network analysis tools, SIEM tools, vulnerability scanners, and defensive systems.
## SIEM Tools (Security Information and Event Management)
SIEM tools collect and analyze logs from multiple sources to detect suspicious activities.They help security analysts monitor systems and respond to incidents.
## Log Management
Logs record system activities and events.Analyzing logs helps in identifying unusual behavior, tracking attacks, and understanding system performance.
## Network Security Tools
These tools monitor and analyze network traffic to detect intrusions or anomalies.They play a key role in protecting data as it travels across networks.
## Firewalls and Defensive Systems
Firewalls act as a barrier between trusted and untrusted networks.They filter incoming and outgoing traffic based on security rules to prevent unauthorized access.
## Programming Languages in Cybersecurity
Programming languages help automate tasks, analyze data, and build security tools.They are essential for both offensive and defensive security roles.
## Common Languages Used
- Python → automation, scripting, analysis
- SQL → database querying and investigation
- Bash/Linux → system operations and scripting
